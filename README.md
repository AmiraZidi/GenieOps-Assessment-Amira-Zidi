## Project Overview

This project was built as part of the **GenieOps frontend assessment**. It demonstrates modern frontend skills.

## Components

- **Hero.jsx:**  
  The landing section of the app. It also includes the Navbar.jsx inside it.

- **Navbar.jsx:**  
  The top navigation bar with links and a dark/light mode toggle.

- **ParallaxSection.jsx:**  
  Handles parallax effects and scroll-based animations. I used Framer Motion to add smooth depth and motion as you scroll.

- **Timeline.jsx:**  
  The main scroll-driven timeline. It tracks scroll progress with Framer Motion’s `useScroll` and triggers animations for each step in Step.jsx.

- **Step.jsx:**  
  Represents each step in the timeline. Steps fade in and translate as you scroll, with glow and visibility controlled dynamically using Framer Motion’s `useTransform`.
